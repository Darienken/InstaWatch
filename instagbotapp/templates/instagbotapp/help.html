多What is InstaWatch?
InstaWatch is a tool developed to save time when watching the public information of an Instagram profile.
Let's say that you want to see the information of a profile, you can do it but you need to follow a set of steps, and if you want to see the information of 1.000 profiles, then you need to do a lot of work, but InstaWatch can solve this problem...
this tool can show you the information from more than 1.000 profiles in only 2 steps.
  1. enter the instagram username
  2. click on "Watch!" button
  3. see the information.
If you want to upload a list of user you must to follow the next steps
  1. choose your excel file
  2. click on "Watch list!" button
  3. download your excel list with all the information ready to watch.
So Easy! right?

多How to upload an excel lists?
In this section I'll explain you how to upload an excel list,

First, you must to create an excel file, be secure that the extension is .xlsx,




Second, you must to write users WITHOUT CAPITAL LETTERS,


under users you must to write usernames that you want to watch


this page will search for the "users" word, so if you don't write users the request will not work and you'll have an error, under users you must to write usernames that you want to watch and you must to write usernames exactly as it appears in Instagram.
